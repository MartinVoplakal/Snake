REDME

CZ
Nastavení programu je uložené v souboru snake.settings.
Pokud si chcete přizpůsobit grafiku, otevřete soubor v libovolném textovém editoru a upravte hodnoty (doporučené editory: PSPad, pozn. blok). POZOR nesmí se změnit číslování řádku. Pro výběr barev můžete použít přiložený .PDF manuál. Vadný soubor s nastavením může způsobit selhání celého programu.
Můžete použít přednastavené soubory s nastavením ve složce Mods.
Aktivujete je přesunutím do hlavní složky a přejmenováním.

EN
The program settings are saved in snake.settings file.
You can change the values in this file and set and personalize the graphics and more.
Please DON'T change a number of rows.
If this file is broken it may cause an errors or total fail of this program.
Keep this on your mind if you want to set up it.
You can also use a pre-sets/modes in Mods folder. (copy to the main folder and rename as snake.settings)
So good luck and enjoy the game.
